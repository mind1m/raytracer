# raytracer
My implementation of https://raytracing.github.io/books/RayTracingInOneWeekend.html


![alt text](https://github.com/mind1m/raytracer/blob/main/image_quality.png?raw=true)

## Build&run

### For debug builds

    cmake -B build
    make build_gen


### For release builds

    cmake -DCMAKE_BUILD_TYPE=Release build
    make build_gen
